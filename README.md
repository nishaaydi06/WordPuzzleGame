
## Word Puzzle Game (Android)

###  Project Overview

Word Puzzle Game is an Android application developed using **Java** in **Android Studio**.
The game challenges users to solve word puzzles by identifying correct words based on given difficulty levels and directions.



### Features

* Easy, Medium, and Hard difficulty levels
* Horizontal and Vertical word directions
* Simple and user-friendly UI
* Interactive gameplay
* Android-based mobile application

## How to Run the Project

1. Clone or download the repository
2. Open **Android Studio**
3. Click **Open an existing project**
4. Select the project folder
5. Let Gradle sync complete
6. Run the app on:

   * Emulator OR
   * Physical Android device
